# Gradle Library

A utility library built with Gradle

Installation
===

Add dependency to build.gradle:

```
dependencies {
    implementation 'com.example:gradle-lib:1.0.0'
}
```

Usage
---

Import and use:

```
import com.example.Util;
Util.doSomething();
```

API Reference
---

### Class: Util

Main utility class

### Method: doSomething()

Does something useful

### Method: doSomethingElse()

Does something else
